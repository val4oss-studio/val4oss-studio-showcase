## ❓ Questions fréquentes

**Le site m'appartient-il ?**
Oui. Le code source vous est livré. En cas de résiliation, vous hébergez le site où vous voulez.

**Pourquoi Next.js et pas WordPress ?**
Next.js = performances natives (Lighthouse ≥ 95), SEO technique impeccable, sécurité sans plugins. Code propre, sans dette technique.

**Pourquoi moins cher qu'une agence ?**
Activité secondaire = pas de frais de structure. Vous payez le travail, pas l'overhead.

**Qu'est-ce que le SaaS val4oss ?**
Un écosystème privé hébergé en Europe : whiteboard (Excalidraw), notes de projet (Affine), forum (Flarum), feedback (Fider). Inclus dans le Plan Premium.

**Que se passe-t-il si je dépasse les heures incluses ?**
Heures supplémentaires facturées 50 €/h, avec accord préalable.

**Modes de paiement ?**
Virement bancaire. Développement : 50 % commande / 50 % livraison. Abonnement : prélèvement mensuel.

**Est-ce que je peux prendre uniquement l'abonnement ?**
Oui, si vous avez déjà un site. À étudier au cas par cas selon la stack technique.
